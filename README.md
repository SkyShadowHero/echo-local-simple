# EchoMusic简单本地音乐

本地音乐播放插件，支持浏览和播放本地音频文件。

## 功能

- 扫描本地文件夹中的音乐文件（MP3/FLAC/WAV 等）
- 自动解析 ID3 标签（标题、歌手、专辑、封面）
- 歌词文件关联（同目录 .lrc/.txt）
- 酷狗在线匹配封面（可选，匿名请求不依赖账号）
- 搜索、排序、文件夹筛选
- 播放历史定位、快速回顶

## 使用

1. 插件设置 → 添加音乐文件夹
2. 侧栏「本地音乐」查看歌曲列表
3. 双击播放，支持搜索和排序

## 设置

- **显示文件夹标签** — 歌曲行右侧显示来源文件夹
- **在线匹配封面** — 通过酷狗搜索补全歌曲封面（大概率导致风控，尽量少用）
